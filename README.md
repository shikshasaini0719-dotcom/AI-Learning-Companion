# 🤖 AI Learning Companion

An interactive AI-powered quiz application built using **Python** and **Streamlit**.

## 🚀 Live Demo
https://ai-learning-companion-shiksha.streamlit.app

## 💻 GitHub Repository
https://github.com/shikshasaini0719-dotcom/AI-Learning-Companion

## ✨ Features

- 👤 Student Registration
- 📚 Multiple Subjects
  - Python
  - AI Basics
  - Machine Learning
- 🎯 Random Quiz Generation
- ⏭️ Skip Questions
- 📊 Performance Analytics
- 🎓 Grade Calculation
- 📚 Topics to Review
- 🏆 Leaderboard
- 💾 Result Storage
- 🌐 Live Deployment using Streamlit Cloud

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Git
- GitHub
- Streamlit Cloud
